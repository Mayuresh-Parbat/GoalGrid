# 🎯 GoalGrid

GoalGrid is a modern goal management and productivity platform designed to help users plan, track, and achieve their personal and professional goals efficiently.

## 🚀 Features

- Create and manage goals
- Track daily progress
- Set milestones and deadlines
- Visual progress tracking
- Goal categories and priorities
- Productivity dashboard
- Responsive design
- User authentication and secure access
- Real-time updates

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JSON Web Token (JWT)

## 🔧 Installation

### Clone Repository

```bash
git clone https://github.com/Mayuresh-Parbat/GoalGrid.git
```

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd backend
npm install
npm run dev
```
